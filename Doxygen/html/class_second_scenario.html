)

Executes the bin-packing algorithm after sorting by descending order of packages and after the sort the transports by ascending order of transports. This function achieves only an approach to the actual optimal profit but is much faster than the knapsack algorithm.

Parameters
packages- all packages stored from the database.
transports- all transports stored from the database.
nonDeliveredPackages- stores the packages that could not been delivered in this scenario.
